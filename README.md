# Fashion Store Mobile App (CROMA)

Industrial-grade streetwear mobile experience built with Flutter.

## ✨ Features

- **Storefront**: Browse collections with a brutalist, industrial aesthetic.
- **Product Details**: High-performance image gallery, size & color selection with real-time stock sync.
- **Favorites**: Sync favorites between web and mobile seamlessly via Supabase.
- **Search**: Collapsible, high-speed search with real-time suggestions.
- **Checkout**: Native Stripe integration with secure payment sheets.
- **Address Management**: Multiple saved addresses with a premium selection UI.
- **Order History**: Track your current and past orders with full logistics details.
- **Internationalization**: Full support for Spanish and English (dynamic switching).
- **Admin Portal**: Integrated management of products, orders, and users.

## 🛠 Tech Stack

- **Framework**: Flutter
- **State Management**: Riverpod (with Annotations/Code Generation)
- **Database/Auth**: Supabase
- **Payments**: Stripe
- **Image Optimization**: Cloudinary
- **Architecture**: Feature-first Clean Architecture

## 🚀 Getting Started

1.  **Clone the repo**: `git clone https://github.com/astralk9999/CROMA_APP`
2.  **Install dependencies**: `flutter pub get`
3.  **Setup Environment**: Create a `.env` file with your `SUPABASE_URL`, `SUPABASE_ANON_KEY`, and `STRIPE_PUBLISHABLE_KEY`.
4.  **Run**: `flutter run`

## 👔 Admin Portal

Access the admin dashboard via the profile menu (if authenticated as admin).
- **Dashboard**: Real-time business metrics (Capital, Reach, Operations, Stock Alerts).
- **Products**: Toggle visibility or remove inventory items.
- **Orders**: Full lifecycle management (Pending, Shipping, Delivered, Cancelled).
- **Users**: View customer profiles and history.
